Sample Web application for Student Housing (UTD)

Functionality to be implemented
-> Search for a Room
-> Create a listing (Room available for rent or Looking for roommates to sign a new lease)
-> Maintain user profile
-> Authenticate using college ID (Planning to implement SSO using college ID)

Front - end module: Housing view

Technologies: React JS(JS library), ES6,
              Redux(Predictable State container),
              Web Pack (Module Bundler),
              Yarn(Package Manager),
              Babel(JS Compiler)
